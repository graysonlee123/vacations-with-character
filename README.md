## Build Setup

```bash
# install dependencies
$ npm i

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project
$ npm run generate

# serve the site locally
$ npm start
```

See [Nuxt.js docs](https://nuxtjs.org) for more information.
